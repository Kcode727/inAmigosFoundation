# InAmigos Foundation Website (Improved Version)

This repository contains an improved, modern, fully responsive, and device-adaptive version of the website for the [InAmigos Foundation](https://inamigosfoundation.org.in/).

This project was built as a gesture of appreciation for the foundation's inspiring social welfare initiatives across India, serving as an aesthetic and technical upgrade of their original web presence.

## Features & Improvements

- **Modern & Premium Design System**: Utilizes a professional, content-focused layout with vibrant brand green color accents (#00CC83), sleek typography (Inter font), and generous spacing for enhanced readability.
- **Strict Grid Alignment**: The home page volunteer form features an adaptive grid aligning the "Area of Interest" selection box neatly to the right of "Full Name" (Row 1), with "Email Address" directly below the name (Row 2).
- **Clean SVG Iconography**: Standardized credential elements on the About Us page using clean outline SVGs, eliminating boxed colored icons or cards with solid backgrounds.
- **Dynamic Stats Icons**: Replaced external images inside the impact stats section on the home page with lightweight inline SVGs styled dynamically with theme colors.
- **Universal Responsiveness**: Optimized code to render perfectly on all devices (mobiles, tablets, notebooks, and large screens). Includes an animated hamburger navigation menu drawer for smaller viewports.
- **Circular Testimonials Carousel**: The causes slider carousel automatically loops back to the start index dynamically when navigating beyond boundaries.

## Setup & Running Locally

1. Clone or download this project folder.
2. Open the root-level `index.html` file in any modern web browser to view the site.
3. The root index file will automatically direct you to `home/index.html` to begin browsing.
